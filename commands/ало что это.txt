чтобы создать команду открытия приложения:

вводите название приложения, пример spotify.exe и оно само ищет, нужно лишь выбор правильного пути в консоли, также нужно добавить команду в commands.yaml, название команды берём по названию файла

P.s. в папке open_exe и close_exe вы должны поменять пути у уже существующих файлов, так как они у нас отличаются


чтобы создать команду открытия ссылок в браузере:

создаём файл -> переименовываем в название команды -> копируем ссылку нужного сайта -> внутри файла вставляем ссылку -> открываем commands.yaml -> пишем названиеи файла и внутри слова для открытия (смотрите пример других команд там же)


Чтобы создать свою команду, как в AutoHotkey, надо взять этот скрипт, положить в commands, превратить в .exe (нажимаете пкм -> compile script) и добавляем команду в commands.yaml, название команды берём по названию бинда .exe
